1. Getting started

WorshipToolkit turns a SongSelect chord chart PDF into an editable, transposable chart you can reshape and export. Everything happens in your browser — nothing is uploaded to a server.

  1. Open the app and either drag & drop a SongSelect PDF onto the drop zone, click Choose File to pick one, click ✏️ Start from scratch to create a blank song, or click 📂 Load set file to resume from a previously exported .wt file.
  2. The app parses the PDF automatically. Multi-song PDFs are split into individual songs.
  3. Two-column layouts and superscript chord extensions (e.g. Fm⁷) are recognized and normalized automatically (e.g. to Fm7).
  4. Direction notes such as To Tag or To Interlude, and bar notation like | Am7 | G |, are preserved as italic annotations under the relevant line.
Only PDF files are supported. If nothing is detected, double-check that the file is a SongSelect-style chord chart PDF, not a scanned image.

2. Your song set

Once a PDF is loaded (or you start from scratch), every song appears in the Songs in Set list on the left (on mobile, a horizontal scrollable tab bar at the top).

  • Click any song in the list to switch the editor to that song.
  • The current key for each song is shown next to its title.
  • Rename a song title — click the song title in the editor toolbar (above the BPM chip) to edit it inline. Press Enter to save or Esc to cancel. This is especially useful for songs added as blank "New Song" entries.
  • Remove a song — hover over a song in the list and click the ✕ button that appears on the right. Removing the last song returns you to the upload screen.
  • Collapse the sidebar — on desktop and tablet, click the ◄ button in the sidebar header to collapse it to a narrow strip, giving more room to the editor. Click ► to expand it again. On mobile, tap Songs in Set ▼ to collapse the song bar and free up vertical space; tap again (Songs in Set ▲) to expand it.

Adding songs to an existing set

On desktop and tablet, + New Song and + Import PDF appear directly at the end of the song list (below the last song). On mobile, tap the + button at the end of the tab strip to reveal them:

  • + New Song — adds a blank song to the bottom of the list. Give it a name, add sections, and build it from scratch.
  • + Import PDF — appends songs from another PDF without replacing what's already in your set. Great for building a multi-song set from separate files.
  • If any imported song title already exists in the set, a warning appears. You can choose to Add anyway (add all including duplicates) or Skip duplicates (only import songs with new titles).

Reordering songs

  • Drag the ≡ handle on any song row up or down to reorder the list. This also works on touch screens — press and hold the handle, then drag.
  • Reordering is undoable with Ctrl/Cmd + Z.

3. Editing chords & lyrics

The editor mirrors the printed layout: chords float above the lyric line they belong to.

Chords

  • Rename a chord — click it, type the new chord, then press Enter (or click elsewhere) to save. Press Esc to cancel.
  • Reposition a chord — click and drag it left or right to line it up exactly where it falls in the lyric.
  • Add a chord — use the + button in the left gutter of a line, or click + chord on an empty line. New chords start as C and open for editing immediately.
  • Remove a chord — hover over it and click the small × that appears.

Lyrics & lines

  • Click directly into any lyric line to edit the text in place.
  • Add a new line under a section with the + Line button in that section's header.
  • Remove a line using the × button in its left gutter. A section always keeps at least one line.

Tempo & time signature

  • Edit tempo — click the BPM chip in the toolbar, type a new number, then press Enter (or click elsewhere) to save. Clearing it shows a + BPM placeholder you can click to set a new value. Press Esc to cancel.
  • Edit time signature — click the time signature chip (e.g. 4/4) next to the BPM chip, type a new signature such as 3/4 or 6/8, then press Enter to save. An invalid entry reverts to the previous value. Press Esc to cancel.

Annotations

  • Add a note — on a lyric line with no annotation, hover it and click + note, type a direction note or bar notation (e.g. (To Tag) or | Am7 | G |), then press Enter to save.
  • Edit a note — click an existing annotation to edit it in place.
  • Remove a note — while editing, click the × next to the annotation input.
  • Annotations are stored untransposed and transposed only for display, so bar notation like | Am7 | G | stays correct no matter how many times you transpose the song.

4. Working with sections

  • Add a section — below the song, use the quick-pick buttons (INTRO, VERSE, CHORUS, PRE-CHORUS, BRIDGE, OUTRO, TAG) or type a custom name and click Add / press Enter.
  • Reorder sections — grab the ⠿ drag handle in a section's header and drop it where you want it.
  • Remove a section — click the ✕ button in the section header.

5. Keys & transposition

  • Step transpose — use the ▲ / ▼ buttons next to the key display to move the whole song up or down one semitone at a time. Every chord on the page updates instantly.
  • Jump to a key — pick any key directly from the Jump to dropdown.
  • Reset — once you've transposed, a Reset button appears showing the original key; click it to snap straight back.
  • Whenever the song is transposed, a banner above the chart confirms the change, e.g. Transposed from G → A (+2 semitones).

6. Bass notes & Nashville numbers

  • 🎹 Bass Notes — toggles every chord down to just its root/bass note (e.g. D/F♯ → D). Handy for simplified or beginner charts.
  • 1 2 3 Nashville — converts every chord to a scale-degree number (1–7, with ♭/♯ prefixes for accidentals), relative to the song's current key. Useful for ear-trained players and key-independent charts.
  • Both toggles can be combined with transposition, and both are respected by every export.

7. Undo & redo

Every edit — chord changes, lyric edits, section moves, additions and removals — can be undone.

  • Use the ↩ Undo / ↪ Redo buttons in the toolbar.
  • Or use the keyboard: Ctrl/Cmd + Z to undo, Ctrl/Cmd + Shift + Z (or Ctrl/Cmd + Y) to redo.
  • History holds up to 50 steps per editing session, and is cleared whenever you load a new PDF.

8. Exporting & sharing

Export buttons live in the song toolbar:

  • ↓ Song PDF — exports just the song you're currently viewing as a clean, two-column, monospace chord chart PDF.
  • ↓ All PDF — exports every song in the set into a single PDF, in list order.
  • ↓ Markdown — exports the full set as a .md file, with chord rows placed above each lyric line, viewable in any Markdown reader.

All exports respect your current transposition, the Bass Notes toggle, and the Nashville toggle, and include any direction/bar-notation annotations, correctly transposed.

Round-trip friendly: PDFs exported from WorshipToolkit can be re-uploaded later. The app embeds the exact column layout in the file so your edits, chord positions, and structure come back intact — no need to keep a separate source file.

9. Saved sets & autosave

A set is your current collection of songs. Saving a set lets you snapshot it and return to it later — even if you close the browser or start working on a different set in the meantime.

The header shows the active set's name (e.g. 💾 Sunday Service ▾), or 💾 Untitled set ▾ if nothing is saved yet. Click it to open the Saved Sets panel. At the top of the panel, the current set name and last-modified time are shown — or Unsaved set if no set is active yet.

Starting a new set

  • Click ➕ New in the header (or + New inside the Saved Sets panel) to clear the workspace and start fresh. On mobile the header button shows just the ➕ icon.
  • If you're currently in a named set (it's already autosaved), the workspace clears immediately.
  • If you have unsaved work, a prompt appears: enter a name to save it first, or click Skip & New to discard and continue.

Saving a set

  • Type a name for your set (e.g. Sunday 29 June) in the input at the top of the panel and click Save (or press Enter).
  • If you save again with the same name, the existing set is updated in place rather than duplicated.
  • Up to 20 sets can be stored. If you exceed this limit, the oldest set is removed.

Autosave to the active set

  • Once a set is saved or loaded, it becomes the active set. Every edit you make — chord changes, transpositions, section moves — is automatically saved back to that set in real time. No need to keep clicking Save.
  • The active set is highlighted with an active badge in the panel, and its name appears in the header button.
  • If you reload the page, the app restores both your songs and the active set so you can continue right where you left off.

Loading a set

  • Each saved set shows its name, save date, and song count. Click Load to restore it — the editor updates immediately and that set becomes the new active set.

Managing sets

  • Rename — click the pencil icon (✎) on any set to rename it inline, then press Enter or click elsewhere.
  • Delete — click the ✕ button to permanently remove a set from storage.
  • Sets are stored only in this browser on this device. Use export/import (below) to move them to another device.

Exporting and importing set files

  • Export (↓) — click the download button on any saved set to save it as a .wt file. This file contains the full song data including all edits, transpositions, and chord positions.
  • Import from the Saved Sets panel — click ↑ Import .wt file at the top of the panel, pick a previously exported .wt file, and the set loads into the editor immediately. It's also saved to your set list so you can return to it later.
  • Import from the upload page — on the landing/upload page, click 📂 Load set file to open a .wt file directly without going through the panel.
  • .wt files are plain JSON and can be shared via email, cloud drive, AirDrop, or any other file-transfer method.

10. Appearance & accessibility

  • Text size — use A- / A+ in the header to scale the whole app's text up or down. Your choice is remembered for next time.
  • Dark mode — click the 🌙 / ☀️ button in the header to switch themes. WorshipToolkit also follows your system's light/dark preference the first time you open it.

11. Tips, autosave & troubleshooting

  • Autosave — your set is continuously saved to this browser's local storage. Refreshing or closing the tab won't lose your work; only starting a new set via ➕ New in the header clears the workspace.
  • Everything is local — no song data leaves your device. Clearing your browser's site data will remove your saved sets.
  • Nothing parsed from my PDF? Make sure it's an actual SongSelect chord chart export, not a scanned/photographed image — the parser reads text, not pixels.
  • Chord didn't land where I dragged it? Positions snap to whole characters so chords never overlap; drop it a little further along the line if it snapped back.
  • Want a clean slate for one song only? You can remove and re-add sections/lines rather than starting a whole new upload.
